# Seif Zakaria's Portfolio  

Welcome to the source code for my personal portfolio! 🎉  
This repository contains all the code and assets used to build my portfolio, showcasing my skills, projects, and experience.  

## Features  
- **Responsive Design**: Optimized for all devices.  
- **Projects Showcase**: Highlights my best work with descriptions and live demos.  
- **About Me**: A glimpse into who I am and what I do.  
- **Contact Form**: Easy way for visitors to reach out.
- **Downloadable Resume**: Easy and fast way to view my pdf resume (for recruiters).  

## Technologies Used  
- **Frontend**: React.js, Next.js, HTML, CSS, JavaScript.  
- **Styling**: TailwindCSS
- **Deployment**: Hosted on Vercel.  

## Setup Instructions  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/seifzellaban/portfolio.git  
   ```  
2. Navigate to the project folder:  
   ```bash  
   cd portfolio  
   ```  
3. Install dependencies:  
   ```bash  
   bun install  
   ```  
4. Start the development server:  
   ```bash  
   bun dev  
   ```  
5. Open your browser and go to `http://localhost:3000`.
